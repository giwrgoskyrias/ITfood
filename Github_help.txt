
Παίρνουμε ο καθένας ένα fork από το βασικό(αντίγραφο). 

Ο καθένας έχει δικαιώματα να αλλάξει το δικό του fork.(αντίγραφο του βασικού δέντρου)

Σε κάθε αλλαγή στο fork μας θα τις συγκρίνουμε με το αρχικό ώστε να ενωθεί.

Κάθε φορά που ξεκινάω ένα καινούριο feature φτιάχνω ένα καινούριο μπράντς(παρακλάδι).

Μόλις το feature είναι έτοιμο, το ενώνω με το βασικό δέντρο(δικό μου μάστερ)


Βήματα

1.https://prnt.sc/ptbr7c --> σε μεταφέρει στο ΔΙΚΟ ΣΑΣ ITfood.
2.κάνετε clone στο visual studio (το δικό σας).https://prnt.sc/ptbt8g και βάζουμε το λινκ από https://prnt.sc/ptbtqg
3.κάνουμε branch https://prnt.sc/ptbuih με όνομα το όνομα του feature.
4.κάνουμε τις αλλαγές ή προσθέτουμε αρχεία και κάνουμε commit. https://prnt.sc/ptbvi2
5.κάνουμε push https://prnt.sc/ptbw06
6.κάνουμε pull request από το github https://prnt.sc/ptbyga στο δικό μου https://prnt.sc/ptbz7g
7.όταν υπάρχουν αλλαγές και θέλουμε να προσθέσουμε features κάνουμε pull https://prnt.sc/ptc1nn
